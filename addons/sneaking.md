## Sneaking
**By: EazyFTW**
<br>

The Sneaking addon gives you the ability to to check if a player is sneaking.
<br>

### Elements
* **Player Is Sneaking** - *Will check if a player is Sneaking.*
<br>

### Constructors
* **Sneak + F-Key Press** - *Will run when a player presses a the Sneak + F-Key.*
